# 画像生成AI用プロンプト (システム構成図ビジュアル化)

このファイルには、ClassForge のシステム構成をビジュアル化するための画像生成AI用プロンプトが記録されています。

---

## 1. アイソメトリック 3D スタイル
システム全体の構造を立体的かつ精密に表現するスタイルです。プレゼン資料などに最適です。

**Prompt (English):**
A high-quality isometric 3D infographic showing a modern web system architecture. Central hub is a sleek cloud server icon (Vercel). To the left, a user laptop and smartphone showing a video learning interface. To the right, a database cylinder (Supabase), a credit card icon (Stripe), and an email icon (Resend). In the foreground, a professional workstation representing local AI video production with Python code and video editing icons. All elements are connected by glowing neon indigo data lines. Clean white background, minimalist design, soft shadows, premium tech aesthetic, 8k resolution, indigo and emerald color palette.

**日本語訳:**
モダンなWebシステムアーキテクチャを示す高品質なアイソメトリック3Dインフォグラフィック。中央には洗練されたクラウドサーバー（Vercel）。左側には動画学習画面が表示されたノートPCとスマホ。右側にはデータベース、決済（Stripe）、メール（Resend）。手前にはPythonコードと動画編集アイコンがあるローカルAI動画制作ワークステーション。すべての要素は光るネオンインディゴのデータラインで繋がっている。クリーンな白背景、ミニマルなデザイン、柔らかな影、プレミアムなテック感、8k解像度。
