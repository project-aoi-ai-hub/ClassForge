
## トラブルシューティング

| 症状 | 対処法 |
|------|--------|
| `prisma generate` がエラー | `DATABASE_URL` の接続文字列を確認 |
| シードが失敗する | DBへの接続確認・既存データの競合がないか確認 |
| ログインできない | シードが完了しているか・パスワードが `admin123` であるか確認 |
| Stripe 購入後に `/thanks` に遷移しない | Stripe CLI の Webhook 転送が起動中か確認 |
| サムネイルアップロードが「アップロードに失敗しました」になる | Supabase Dashboard の Storage に `course-thumbnails` バケットが**作成済みかつ Public**であることを確認。環境変数の設定だけでは不十分。 |
| D&D が反応しない | ドラッグハンドル（左端の ⠿ アイコン）を掴んでいるか確認。編集中のセクションは掴めないので「キャンセル」後に試す |
| 完了マークボタンが表示されない | 視聴ページの URL が `/learn/[courseSlug]/[lessonSlug]` の形式になっているか確認 |
| パスワード変更でエラーになる | 新パスワードは「英大文字・数字・記号を各1文字以上含む8文字以上」の要件を満たすこと（例: `NewPass123!`） |
