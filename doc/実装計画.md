## DB スキーマ（確定）

| モデル | 役割 |
|--------|------|
| `User` | 会員（Role: USER / ADMIN）|
| `Account` / `Session` / `VerificationToken` | Auth.js 用 |
| `Course` | 講座（status: DRAFT / PUBLISHED）|
| `Section` | 講座内の章（sortOrder で並び順管理）|
| `Lesson` | レッスン（isPreview / videoUrl / content / duration）|
| `Purchase` | 購入履歴（Stripe セッション ID / status: PAID / REFUNDED）|
| `Progress` | 視聴進捗（watchedSeconds / isCompleted / lastWatchedAt）|
